# SYKO Design System (v1)

## Color Tokens

- Background primary: `#040404`
- Panel background: `#0f0f10`
- Foreground primary: `#f5f5f4`
- Foreground muted: `#a1a1aa`
- Brand accent red: `#e63434`

## Typography

- Brand display (`SYKO` and hero subword): `Bebas Neue`
- Body and UI copy: `Inter`
- Tracking:
  - Brand wordmark: wide tracking for premium editorial tone
  - Body: normal tracking for readability

## Layout Rhythm

- Section shell max width: `80rem`
- Section vertical spacing: `4.5rem`
- Card radius: `1rem - 1.25rem`
- Thin contrast border: `rgba(255,255,255,0.10)`

## Component Rules

- Navigation: high-contrast sticky on blur panel
- Hero: split architecture with explicit dual-entry CTAs
- Cards: subtle border + dark layered backgrounds
- Buttons:
  - Primary: red brand fill
  - Secondary: outlined on dark

## Cultural Fusion Rule

- Use cultural motifs as subtle structural texture only.
- Avoid literal decorative overload.
- Prioritize clean modern hierarchy with selective heritage accenting.
