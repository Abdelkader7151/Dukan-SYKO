# SYKO Wireframe Specification

## Home

1. Header (sticky nav)
2. Split hero:
   - left: brand statement + SYKO typography
   - right: visual card with two entry actions
3. Shared identity strip
4. Restaurant and Dukan preview cards
5. Footer

## Restaurant Page

1. Hero block
2. Menu focus card list
3. Visit/contact card

## Dukan Page

1. Hero block and logo media panel
2. Category highlight grid

## Story Page

1. Story hero and framing copy
2. Animated timeline cards
3. Map story panel with high-quality PDF access

## Visit Page

1. Shared visit hero
2. Restaurant contact card
3. Dukan contact card

## Mobile Priorities

- Above-fold CTA visibility
- Fast route transitions
- Legible typography on dark backgrounds
- Contact actions visible without excessive scroll
